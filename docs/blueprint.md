# **App Name**: Uncoverly

## Core Features:

- Hero Section: Display the 'Uncoverly' name prominently with the slogan 'Ingl
ess que se adapta a ti. No al rev\[00e9]s.' below it.
- About Section: Present a concise 'About Us' section highlighting Uncoverly's personalized approach to English education in a visually appealing card format.
- Pricing Plans: Showcase the three class packages (private, small group, and group) in a simple, easy-to-compare card layout.
- Interactive Calculator: Develop an interactive calculator that computes monthly course costs based on class type and frequency selected by the user.
- WhatsApp Contact Button: Implement a floating WhatsApp button for immediate customer support access.
- Robot Mascot Integration: Incorporate the robot mascot subtly across the webpage, using vector illustrations and consistent styling, using illustrations in the header and beside the calculator widget. The illustrations for the robot will have placeholders.
- Light/Dark Mode Toggle: A light/dark mode toggle will allow switching between a light color theme and a dark color theme.
- Featured Courses Section: Showcase featured courses with brief descriptions and a 'Descubre' button that navigates to the pricing plans.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trust and approachability.
- Background color: Light gray (#F5F5F5) for a clean and modern feel in light mode; Dark gray (#333333) in dark mode.
- Accent color: Muted green (#84D9B3) to highlight important interactive elements.
- Font pairing: 'Poppins' (sans-serif) for headings to convey a geometric modern feel, and 'PT Sans' (sans-serif) for body text to enhance readability.
- Use clean and friendly icons, such as those from the Material Design set. Icons should consistently be in the muted green accent color.
- Maintain a responsive layout that adapts seamlessly to both mobile and desktop views.
- Incorporate subtle transitions and animations for a smooth and engaging user experience.
